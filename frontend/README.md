# Parking Management (Flutter Frontend)

This is a minimal Flutter frontend for a parking management app. It includes a login screen, a dashboard, and a parking slots list with simple reserve/release functionality.

Run locally:

1. Install Flutter: https://flutter.dev/docs/get-started/install
2. From this folder run:

```bash
flutter pub get
flutter run
```
